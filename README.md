# Door Reduction

Node.js backend for door reduction calculations and export output generation.

## Prerequisites

- Node.js 18+ (recommended)
- npm

## Setup

```bash
npm install
```

## Run

```bash
npm start
```

For development with auto-reload:

```bash
npm run dev
```

## Project Structure

- `src/`: core calculation and export logic
- `data/`: CSV source data
- `public/`: static assets and Zoho widget files
- `server.js`: API server entry point
